AyurSutra is a full-stack AI-powered conversational platform that provides personalized Ayurveda-based health guidance. It leverages modern web technologies and large language models to deliver intelligent, context-aware responses for holistic wellness.

🔗 Live Demo: https://ayursutra-two.vercel.app/

Features:

• Conversational AI Interface
Interactive chat-based UI for seamless user experience.

• AI-Powered Responses
Generates personalized Ayurveda recommendations using LLMs.

• Context-Aware Guidance
Provides meaningful responses based on user input and health context.

• Fast & Responsive UI
Built with React (Vite) for high performance and smooth interactions.

• Full-Stack Architecture
Backend APIs handle request processing, logic, and integrations.

• Deployed on Vercel
Ensures scalability, fast load times, and global accessibility.
